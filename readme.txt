== Description ==

Isvek-Theme на Bootstrap 5 - это тема, которая использует фреймворк Bootstrap 5 для создания адаптивного и мобильного дизайна.

Bootstrap 5 - это популярный фреймворк для разработки веб-сайтов, который предлагает множество готовых компонентов и стилей для создания современных и красивых сайтов.

Тема может включать в себя все преимущества этого фреймворка, такие, как адаптивная сетка, готовые компоненты для навигации, форм, кнопок и многого другого.

== Installation ==

= Изнутри WordPress =
1. Перейдите в меню "Внешний вид > Темы > Добавить новую".
2. Найдите "Isvek".
3. Установите и активируйте.
